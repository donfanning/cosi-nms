Throughputter

Modem Throughput Applet


  • Throughputter is a Java Applet intended to be used with an analog modem. It estimates the transfer rate between your modem and the server where this applet lives by retrieving two files over http. These reported rates should be lower than your connection rate. For example, a 40kbps throughput with a 56k modem is normal, since some of the speed is "lost" to protocols.

  • Java must be enabled to use this applet. In Netscape, this option is under Edit | Preferences | Advanced | Enable Java.

  • If you see a message like: 128kbps is too high for a modem, and you are using an analog modem, then this is most likely caused by Netscape's cache. Just clear the cache by going under Edit | Preferences | Advanced | Cache, and clicking on Clear Memory Cache and Clear Disk Cache.
