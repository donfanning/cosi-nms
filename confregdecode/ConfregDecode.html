IOS(tm) Config-Reg Decoder

This is a new version of the configuration register decoder tool. This new one is also written in Java , and requires Netscape 4.5 or higher or Internet Explorer 4.0 or higher. This new version supports the newer 3600, 2600, and MC3810 routers, plus adds some more decoding/encoding features (like NetBoot files). The notes section simply tells some interesting register-related info on each router series. The '*' next to certain items indicate they are Cisco default. Please email me with any comments or problems.

You can enter the config-register in the form of 2102, 0x2102, or 0X2102

This applet requires Netscape 4.5 or higher with Java turned on.
